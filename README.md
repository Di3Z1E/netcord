## ⚡ Overview

**NetCord** is a streamlined, professional-grade IPv4 Network Manager. Designed to replace the aging Windows network stack interface, it provides a high-visibility dashboard for monitoring and reconfiguring network adapters with zero friction.

### 🔥 Key Features

- 📊 **Unified Dashboard**: Real-time telemetry for active adapters and public IP verification.
- ⚡ **Rapid Configuration**: Instant switching between **DHCP** and **Static IP** assignments.
- 🔗 **Advanced Addressing**: Support for secondary IP mapping on single interfaces.
- 🧪 **Diagnostic Suite**: Native Ping testing, DNS cache purging, and lease renewals.
- 📂 **Profile Engine**: Save environment-specific network states for instant deployment.
- 🌑 **Sleek UI**: A clean, modern dark-mode interface designed for power users.

## 📥 Installation

> [!IMPORTANT]
> **Elevated Privileges** are required to modify system-level network configurations.

1. **Clone the Repository**
```bash
   git clone https://github.com/Di3Z1E/netcord.git
   cd NetCord
```

2. **Environment Setup**
```bash
   pip install -r requirements.txt
```

3. **Execute**
```bash
   python main.py
```

---

## 📦 Distribution

To compile a standalone binary for Windows, use the included specification file:

```bash
pyinstaller netcord.spec
```

The executable will be generated in the `dist/` directory.
