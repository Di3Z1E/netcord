# Project Media Assets

This folder contains visual assets for the NetCord project, such as:
- **Project Logo**: (`logo.png` / `logo.ico`)
- **Project Banner**: (`banner.png`)
- **Screenshots**: High-resolution previews of the application.

### Usage in README
To use these assets in your `README.md`, you can reference them using relative paths:
```markdown
![NetCord Logo](media/logo.png)
```
