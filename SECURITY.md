# Security Policy

## Supported Versions

Currently, only the latest version of NetCord is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| < 1.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within NetCord, please do not open a public issue. Instead, please report it via GitHub's private vulnerability reporting or by contacting the maintainers directly.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
